# Access Control

В качестве модели управления доступом используется **мандатная модель** (Mandatory Access Control) по принципу *Bella-LaPadula*. Также проверяется доступ относительно **целостности** (integrity) по принципу *Biba*.

### Ключевые понятия

- **Конфиденциальность** (confidentiality): число (неотрицательное, 8-ми битное). Определяет уровень конфиденциальности субъекта или объекта.
    - *Чтение* разрешено, если конфиденциальность субъекта >=  конфиденциальности объекта
    - *Запись* разрешена, если конфиденциальность субъекта <=  конфиденциальности объекта
- **Категории** (category): число (неотрицательное, 64-х битное). Определяет битовую маску (bitmask) категорий, в которые входит субъект или объект.
    - *Чтение* разрешено, если в категории субъекта включены все категории объекта
    - *Запись* разрешена, если в категории объекта включены все категории субъекта
- **Уровень целостности** (integrity): число (неотрицательное, 32-х битное). Определяет уровень целостности субъекта или объекта.
    - *Чтение* разрешено, если целостность субъекта <= целостности объекта
    - *Запись* разрешена, если целостность субъекта >= целостности объекта
- **Возможности** (capabilities): число (неотрицательное, 64-х битное). Определяет битовую маску (bitmask) специальных возможностей, которые имеет субъект или может требовать объект для доступа.
    - *Чтение* и *запись* разрешены, если субъект имеет все требуемые объектом специальные возможности
